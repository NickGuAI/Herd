# Domain Distill

Use this skill when raw research should become reusable knowledge.

Extract:

- Stable facts.
- Techniques and patterns.
- Opinionated beliefs or hypotheses.
- Open questions.
- Source links for audit.

Do not mix unverified speculation with facts.
