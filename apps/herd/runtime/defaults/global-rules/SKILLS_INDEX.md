# Skills Index

Use this file to route recurring work to installed skills before inventing a new
procedure.

Default skill routing:

- `commander-create-wizard`: create or configure commanders through backend APIs.
- `create-automation`: create scheduled, manual, or quest-triggered automations.
- `commander-memory-cleanup`: consolidate commander memory files.
- `context-rot-cleanup`: inspect and clean stale durable context.
- `engineering-review`: review code, architecture, issues, or pull requests.
- `legion-investigate`: investigate a bug and produce a root-cause issue plan.
- `legion-implement`: deliver a scoped implementation with verification.
- `write-new-skill`: create or improve a reusable skill.

Add local skills here only when the routing rule will help future commander work.
