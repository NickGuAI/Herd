# User Context

This file is owned by the Herd operator. Use it for durable facts about the
operator's preferences, current focus, and ambiguity-resolution rules.

Keep this file short and high-signal:

- Identity or preference facts that should affect most commander work.
- Current strategic focus that should route priorities.
- Rules for resolving ambiguous requests.

Do not store secrets, access tokens, raw private logs, or routine task history
here. Put project-specific guidance in the closest project guide instead.
