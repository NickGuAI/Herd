# Create Deck

Use this skill when a user wants research or planning material converted into a deck.

Workflow:

1. Confirm the target audience and purpose.
2. Extract the core message, sections, and evidence.
3. Produce a slide outline with title, key points, visuals, and speaker notes.
4. Call out missing assets or facts before inventing them.

Do not create external files or publish content unless the user asks for that action.
