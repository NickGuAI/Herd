# Daily Briefing

Use this skill for morning or planning briefs.

Include:

- Today's schedule and important constraints.
- Top priorities.
- Open loops that need follow-up.
- Draft decisions or messages the user should review.

Ask before sending emails, changing calendar events, or publishing anything externally.
