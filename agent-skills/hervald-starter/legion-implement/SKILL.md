# Implementation Delivery

Use this skill for implementation work that has a clear issue or delivery brief.

Steps:

1. Confirm scope and current branch.
2. Read the relevant code and docs.
3. Implement the narrowest complete fix.
4. Run targeted verification and broader checks when risk requires them.
5. Summarize changed files, verification, and remaining risks.

Do not merge or deploy unless the user explicitly asks.
