# Knowledge Search

Use this skill when prior notes, reports, beliefs, or archives may already contain the answer.

Workflow:

1. Search the configured knowledge roots.
2. Read the most relevant source files directly.
3. Separate sourced facts from inference.
4. Link or cite the source paths in the answer.

If the knowledge index is unavailable, fall back to scoped file search and say so.
