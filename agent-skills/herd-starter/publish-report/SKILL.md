# Publish Report

Use this skill when a completed report needs to be packaged for publishing.

Before publishing:

- Confirm the intended audience.
- Check that sources and images are safe to publish.
- Produce a concise title, summary, and final artifact path.
- Ask for explicit approval before external publication.
